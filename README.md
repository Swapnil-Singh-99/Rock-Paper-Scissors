## Run Locally

Clone the project

```bash
  git clone https://github.com/Swapnil-Singh-99/Rock-Paper-Scissors.git
```

Go to the project directory

```bash
  cd Rock-Paper-Scissors
```


Start the Live Server on VS Code

```bash
  Open with Live Server
```
